NotebookLM セットアップ用プロンプト

会社PCの Claude Code に貼り付けるプロンプトです。受け取った notebooklm_std フォルダを開いた状態で使ってください。

このフォルダの notebooklm_raw.ipynb のセットアップ手順どおりに環境を準備し、捨てノートブックで動作確認して。NotebookLM の操作はこのフォルダの nlm_client.py だけを使う（他の NotebookLM の MCP・ツールは併用しない）。
補足: nlm_client.py の88行目付近のエラーメッセージ内 "Invoke-NotebookLM login" は古い表記で、正しくは "python nlm_client.py login"。

補足

  • login 時は会社の Google アカウントでログイン（会社の NotebookLM 契約を使う）。
  • 必要環境：Python 3.10 以上 / pip install httpx playwright / Google Chrome。
  • 社内のネットワーク制限（プロキシ・pip 制限・Chrome デバッグ起動のブロック）で止まる場合あり。その時は管理者/IT に確認。
